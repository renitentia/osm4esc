osm4esc
=======

Steps to participate:
	* create account on github.com
	* admin must add username to Collaborators
	* checkout repository: git clone git@github.com:species/osm4esc.git
	* install the Java OpenStreetMap editor from http://josm.openstreetmap.de/wiki/Download
	* load file central.osm.xml in JOSM
	* edit, save
	* git commmit
	* git push


steps for further contributing
	* git pull
	* edit,save
	* git commit 
	* git push
